*不建议在移动端使用jQuery*

> jQuery做了很多桌面浏览器的兼容性问题，特别是IE，但是移动端没有 IE 浏览器
> 主流浏览器：谷歌、火狐(2016年停止了维护和更新)、Safari、百度、360、qq ,内核基本上都是webkit或blink, 兼容 `-webkit`
> 建议使用H5的api ,或者使用 zepto.js 的库（基于高版本浏览器开发）
